# 🎯 Facial Recognition Attendance System Using Deep Learning

A smart **attendance management system** that automatically marks attendance using **facial recognition technology**.
The system uses **deep learning and computer vision** techniques to detect and recognize faces in real time and record attendance accurately.

This project eliminates the need for manual attendance and helps prevent proxy attendance by verifying each individual through facial recognition.

---

## 🚀 Features

* Real-time face detection using webcam
* Face recognition using deep learning models
* Automatic attendance marking
* Attendance stored in database/file
* User registration with face data
* Live camera feed processing
* Secure and accurate identification
* Simple and easy-to-use interface

---

## 🧠 Technologies Used

* **Programming Language:** Python
* **Libraries:** OpenCV, Face Recognition, NumPy
* **Machine Learning:** Deep Learning based face encoding
* **Interface:** Python GUI / Web Interface
* **Database:** CSV / MySQL (for attendance records)

---

## ⚙️ How It Works

1. Capture face images of users and store them in the dataset.
2. Train the system to recognize faces using deep learning encodings.
3. Start the attendance system using a webcam.
4. The system detects and recognizes faces in real time.
5. Attendance is automatically recorded when a recognized face appears.

## 🛠️ Installation

1. Clone the repository

```
git clone https://github.com/yourusername/facial-recognition-attendance-system.git
```

2. Install required libraries

```
pip install -r requirements.txt
```

3. Run the attendance system

```
python attendance_system.py
```
